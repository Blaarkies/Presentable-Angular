LZMA
  • Compresses the entire works of Shakespeare down to 28% of it's original size
  • Available through 7-Zip
  • Uses a dictionary compression scheme
  • Features a high compression ratio
  • Compression-dictionary size is variable in size up to 4GB
  • Encodes strings of characters as elements, using a "copy-paste" approach where previous entries are referenced instead of writing the primitive values down
  • This previous referencing is also encoded, so that frequently used references are recorded in smaller byte values than typically needed
gzip
  • Compresses the entire works of Shakespeare down to 39% of it's original size
  • Used in HTTP compression, to speed up sending HTML
  • A combination of LZW and Huffman coding
  • Licenced under GNU General Public License
  • Initial release, 31 October 1992
  • http://www.gzip.org/
  • Recommended because Internet Explorer does not implement the zlib format correctly
  • AdvanceCOMP and 7-Zip can produce gzip-compatible files
